### Cards

## Interaction States

### Card States

1. **Default:**
   - Border: 2px solid #8FABD4
   - No shadow
   - No transform

2. **Hover:**
   - Border: 2px solid #4A70A9
   - Shadow: 0 10px 25px rgba(74, 112, 169, 0.2)
   - Transform: translateY(-5px)
   - Transition: 0.3s ease

3. **Active (Click):**
   - Transform: translateY(-2px)
   - Brief press effect

4. **After Click:**
   - Navigate to upload view
   - Smooth transition

## File Type Detection

When "Auto-Detect" is selected:
1. User uploads file
2. System analyzes file structure
3. Detects file type automatically
4. Shows detected type in validation results
5. Proceeds with appropriate validation rules

---

## Success Flow

```
1. User sees cards
2. Clicks "Punchrecord"
3. Upload page opens
4. Drags file
5. File uploads (progress bar)
6. Validation runs (progress bar)
7. Results shown (stats cards)
8. Preview table displayed
9. User clicks "Confirm Import"
10. Import completes
11. Success message
12. "Upload Another" button
13. Returns to card selection
```
